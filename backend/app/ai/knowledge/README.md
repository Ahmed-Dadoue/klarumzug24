This directory contains the active multilingual FAQ knowledge files for Dode.

Active files:
- `faq_de.json`
- `faq_en.json`

Archived legacy files:
- `backend/archive/ai_knowledge/faq.json`
- `backend/archive/ai_knowledge/faq_deprecated.json`

Notes:
- The active AI retrieval flow only loads `faq_de.json` and `faq_en.json`.
- Keep new FAQ changes in `faq_de.json` and `faq_en.json` only.
- Older single-language FAQ backups were moved to `backend/archive/ai_knowledge/`.
